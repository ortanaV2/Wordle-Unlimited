## Including
- **dataset**  *(dataset.json)* -  list of words
- **dataset-extender**  *(add_word.py)*  -  program to add words to dataset
- **user-data**  *(userdata.json)*  -  saves the user's progress
- **wordle**  *(wordle.py)*  -  the wordle game
